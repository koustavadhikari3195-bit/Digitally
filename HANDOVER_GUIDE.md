# Digitally Handover Guide

Welcome to the **Digitally** project! This document outlines everything you need to successfully manage, deploy, and maintain the application for your client.

---

## 1. Project Overview
**Digitally** is a dual-purpose web platform designed for a digital marketing agency with an integrated "AI Lab" for client attraction.

### Key Features:
- **Dual-Mode Interface**: Instantly switch between "Agency Services" and "AI Lab Tools".
- **AI Resume ATS Tool**: Analyzes PDFs for ATS compatibility and provides AI-powered bullet point rewriting.
- **Website "Roast" Tool**: Engages users with AI-driven website critiques.
- **Lead Qualifier**: Automatically qualifies potential clients and generates custom growth plans.
- **Admin Portal**: Secure dashboard to manage leads generated by the AI tools.

---

## 2. Technical Stack
- **Frontend**: React (Vite), Framer Motion (Animations), Tailwind CSS (Styling).
- **Backend**: Node.js, Express, MongoDB (Data Storage).
- **AI Engine**: OpenRouter API (Access to top-tier LLMs).
- **Security**: JWT Authentication, Helmet, and CORS protection.
- **Optimization**: Granular code splitting for < 1s initial load speed.

## 3. Deployment (Production)
The project is set up for **Zero-Touch Deployment** using Google Cloud Run and GitHub Actions:

1.  **Infrastructure**: Use [setup_gcp.ps1](file:///e:/Digitally/setup_gcp.ps1) to provision Google Cloud resources.
2.  **GitHub Actions**: The [deploy.yml](file:///e:/Digitally/.github/workflows/deploy.yml) workflow automatically builds and deploys the app on every push to `main`.
3.  **Environment Variables**: Ensure the following are set in the GitHub Registry/GCP Secrets:
    - `MONGO_URI`: Your MongoDB Connection string.
    - `JWT_SECRET`: A secure random string for tokens.
    - `OPENROUTER_API_KEY`: API key from OpenRouter.

4.  Set **Start Command** to `npm start`.

### **No Credit Card? (Use Vercel):**
If Render asks for a card, use [Vercel.com](https://vercel.com).
1.  Connect your GitHub repo.
2.  Vercel will detect my `vercel.json` automatically.
3.  Add your `MONGO_URI` and `OPENROUTER_API_KEY` in the Environment Variables tab.
4.  Click **Deploy**.

## 4. Local Development & Maintenance
To run the project locally for testing or updates:
- **One-Click Start**: Run the [run_app.bat](file:///e:/Digitally/run_app.bat) file.
- **Manual Start**:
    - Backend: `cd backend && npm run dev`
    - Frontend: `cd frontend && npm run dev`

### Updating Content:
- **Branding**: Update `logo.png` and `hero-bg.png` in `/frontend/public`.
- **AI Logic**: Adjust prompts in [aiRoutes.js](file:///e:/Digitally/backend/routes/aiRoutes.js) to fine-tune the "Roasts" or "Qualifiers".
